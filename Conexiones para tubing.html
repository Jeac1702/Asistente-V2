Conexiónes para Tubing

Barril de plástico para conexión de compresión parker

Barril accesorio de compresión de baja presión compacto premontado. De copolímero de acetal exclusivo tiene una resistencia superior para resistir el arrastre y el estrés causado por la compresión.
►Medida 1/4 pulg
►Presión 150 psi
►Temperatura de -17 a 65 °C
 
 
Codo 90° macho para tubing parker

Codo macho 90° para tubing, accesorio de compresión compacto premontado, tiene una resistencia superior para resistir el arrastre y el estrés causado por la compresión. Las tuercas Poly-Tite girarán
alrededor del mango mientras se tensa para evitar la torsión y el debilitamiento del tubo de plástico.
►Conexiones tubo 1/4 x rosca 1/8 pulg
►Presión máxima 150 psi tubo termoplástico y 300 psi tubing metalico
 
 
Conector unión macho para tubing parker

Conector unión tipo macho accesorio de compresión compacto premontado, tiene una resistencia superior para resistir el arrastre y el estrés causado por la compresión. Las tuercas Poly-Tite girarán 
alrededor del mango mientras se tensa para evitar la torsión y el debilitamiento del tubo de plástico
►Conexión tubo 1/4 x rosca 1/16 pulg
 
 
Unión conexión compresión para tubing parker

Unión para tubing para su uso con tubos de metal blando para presiones de hasta 300 PSI y polytubing para presiones de hasta 150 PSI. La funda de acetal cautiva proporciona una elasticidad excepcional 
que resiste el arrastre y el estrés causado por la compresión.
►Conexión de 1/2 pulg
►Fabricado en latón
 
 
Unión prestolok conexión rápida para tubing parker

Unión de latón niquelado prestolok para conexión y desconexión rápida, mediante la tecnología push-to-connect, para baja presión
►Conexiones 1/4 x 1/4 pulg
►Presión de 300 psi
